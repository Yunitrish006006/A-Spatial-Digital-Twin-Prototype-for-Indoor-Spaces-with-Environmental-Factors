# 論文對應說明

本專案是針對下列論文方向所實作的最小完整研究原型：

**基於有限角落感測器與連續影響場估計之室內環境數位孿生模型**

## 研究問題

1. 如何用有限數量的角落感測器估計單房間內的溫度、濕度、亮度分布？
2. 如何在不做 CFD 級模擬的前提下，描述冷氣、窗戶與照明對空間各區域的影響？
3. 如何在有限感測資料下，對候選設備動作給出具可解釋性的排序建議？

## 對應章節

### 第一章 研究背景與問題定義

- 使用 `README.md` 的研究動機與系統輸入輸出描述。
- 強調本研究主張：
  - 單房間
  - 8 顆角落感測器
  - 三因子場估計
  - 設備影響預測
  - 決策推薦而非閉環控制

### 第二章 文獻探討

可對照本專案中的方法切入四個方向：

- 室內環境數位孿生
- 空間插值與場估計
- 智慧建築多感測器融合
- Rule-based / model-based 決策推薦

### 第三章 系統與數學模型

可直接引用本專案的數學結構：

```text
F_v(x, y, z, t) = B_v(x, y, z) + sum(I_j,v(x, y, z, t)) + C_v(x, y, z)
```

其中：

- `v` 為溫度、濕度或亮度
- `B_v` 為背景場
- `I_j,v` 為第 `j` 個設備對變數 `v` 的影響函數
- `C_v` 為由 8 顆角落感測器殘差回歸得到的 trilinear 校正場

### 第四章 數位孿生推估與決策方法

- `digital_twin/model.py`
  實作場生成、感測器估計與校正
- `digital_twin/recommendations.py`
  實作以目標區域舒適度為導向的候選動作排序

### 第五章 模擬案例與結果分析

可直接使用 `digital_twin/scenarios.py` 內建的標準情境：

- 無設備
- 僅冷氣
- 僅開窗
- 僅照明
- 冷氣 + 窗戶
- 窗戶 + 照明
- 冷氣 + 照明
- 三者同時作用

建議分析：

- 場重建誤差
- 區域平均誤差
- 校正前後感測器誤差
- 決策排序與場景一致性

### 第六章 結論與未來工作

可延伸方向：

- 實體 ESP32 佈署與校正
- 加入真實量測資料
- 更細緻的濕度模型
- 多房間耦合
- 自動控制策略

## 論文中需明說的限制

- 本模型是簡化動態模型，不是高精度熱流體模型。
- 濕度為次核心變數，只做一階耦合。
- 感測器配置固定為 8 顆角落，不研究最佳化佈署。
- 驗證以模擬與合理物理假設為主。
