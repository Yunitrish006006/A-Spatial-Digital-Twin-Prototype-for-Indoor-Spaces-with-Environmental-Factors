# 碩士口試倒排待辦表

本文件是給自己勾選用的一頁式版本。正式規定請仍以：

- [畢業條件與學位考試 Checklist](/Volumes/DataExtended/school/docs/admin/graduation_requirements_checklist_zh.md)
- [口試實戰指南](/Volumes/DataExtended/school/docs/admin/oral_defense_playbook_zh.md)

為準。

## D - 60 到 D - 45

- [ ] 和指導教授確認可以申請口試。
- [ ] 確認學分、書報討論、進階書報討論都已滿足。
- [ ] 確認本學期沒有換指導教授，或已滿冷卻期限。
- [ ] 確認畢業條件證明是否已具備，例如研討會證明或系統展示證明。
- [ ] 開始找口試委員，先確認委員名單與資格。
- [ ] 若有校外或非校內委員，先確認是否需要額外資格證明。
- [ ] 完成主要實驗、圖表與論文初稿第一版。

## D - 45 到 D - 30

- [ ] 把論文初稿交給指導教授修改。
- [ ] 固定中英文論文題目，避免後續文件標題不一致。
- [ ] 完成學術倫理課程與修課證明。
- [ ] 跑論文原創性比對。
- [ ] 將相似度控制在安全範圍內。
- [ ] 若結果偏高，先修正文句與引用，不要拖到申請前一天。
- [ ] 初步確認口試日期、時段與場地。

## D - 30 到 D - 21

- [ ] 確認口試委員是否都可出席。
- [ ] 確認校外委員比例符合規定。
- [ ] 準備申請文件：
- [ ] 論文封面
- [ ] 學位考試申請書
- [ ] 論文口試委員建議表
- [ ] 無違反學術倫理聲明書
- [ ] 原創性比對結果
- [ ] 學術倫理修課證明
- [ ] 畢業條件證明
- [ ] 需要簽名的地方都先找老師簽完。

## D - 14

- [ ] 正式送出學位考試申請。
- [ ] 論文初稿寄給口試委員。
- [ ] 若系辦有要求紙本，確認已送到。
- [ ] 向系辦確認聘函、停車、成績表、費用單據的領取方式。

## D - 7 到 D - 3

- [ ] 完成口試簡報短版與長版。
- [ ] 測試簡報中的字型、圖片、動畫與影片。
- [ ] 準備口試當天要印的文件：
- [ ] 資工系論文口試結果表
- [ ] 研究生論文口試評分表 × 委員人數
- [ ] 碩士論文審定書
- [ ] 自備筆數支
- [ ] 準備紙本論文或簡報備份。
- [ ] 確認委員交通、停車與聯絡方式。
- [ ] 若要準備禮品、飲品、點心、餐盒，此時就先下訂。

## D - 1

- [ ] 到系辦領取聘函、成績表、費用簽名表與停車相關資料。
- [ ] 再次確認場地、投影設備、轉接頭、延長線、簡報檔。
- [ ] 確認口試文件標題一致。
- [ ] 若審定書題目仍可能修改，先和指導教授確認簽名策略。
- [ ] 準備水、飲品、點心與餐食。

## D-Day

- [ ] 提前到場測試設備。
- [ ] 擺好委員文件、論文、筆、飲品與點心。
- [ ] 發送聘函給口試委員。
- [ ] 口試結束後，確認所有表單都已簽完。
- [ ] 把結果表、評分表、審定書、成績表、費用單據送回系辦。

## D + 1 到 D + 7

- [ ] 寄訊息或 Email 感謝口試委員與指導教授。
- [ ] 整理委員修改意見。
- [ ] 修改論文與簡報。
- [ ] 若題目有更改，更新封面、審定書與最終論文題目。
- [ ] 視需要將修改後論文寄給委員確認。

## D + 7 到 D + 14

- [ ] 完成論文定稿。
- [ ] 完成圖書館論文上傳與複審。
- [ ] 印製平裝本。
- [ ] 繳交系辦、圖書館、註冊組所需論文。
- [ ] 辦理離校系統或紙本離校手續。

## 本專案專用口試展示檢查

- [ ] 中文論文 PDF
- [ ] 30 分鐘口試版簡報
- [ ] Web demo 可正常啟動
- [ ] 架構圖與驗證流程圖可直接展示
- [ ] 至少 2 到 3 個代表性情境結果
- [ ] 若被問技術細節，可展示 MCP、Gemma bridge、3D preview 或 window direct input
